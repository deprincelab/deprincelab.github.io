Translational Motion


Overview


This tutorial develops solutions to the time-independent Schrödinger equation for several translational problems (e.g., particle-in-a-box, tunneling through a one-dimensional barrier, etc.). Relevant Python code is provided to assist in visualizing the wave functions and obtaining numerical solutions to the time-independent Schrödinger equation.

You can download the Jupyter notebook for this project here.
