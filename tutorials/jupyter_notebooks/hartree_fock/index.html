The Hartree-Fock Self-Consistent Field Procedure


Overview


In this project, we will write an unrestricted Hartree-Fock self-consistent field (SCF) solver in Python. Technically, we will solve the Roothaan equations, which are a representation of the Hartree-Fock approach within a basis of non-orthogonal orbitals. We will obtain all of the required integrals over these orbitals from the Psi4 electronic structure package.

For a tutorial that does not require the use of Psi4, see Daniel Crawford's SCF project. In that project, all required integrals are read from text files so that your code can be written independently of any electronic structure package.

For additional information about the SCF procedure, I recommend Szabo and Ostlund's book, Modern Quantum Chemistry, or David Sherrill's online notes.
